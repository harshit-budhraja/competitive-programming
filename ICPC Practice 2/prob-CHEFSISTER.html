Chef loves playing with her sister a lot. Once her sister invented an interesting game. They bough n balloons from the market and arranged them in a line from left to right. You are given the configuration of the baloons by a string s of length n. Now, Chef's sister wants her to burst the balloons using two devices. For the first device, he can selected at most K balloons of same colour and burst them up. For the second device, he can selected at most K balloons all of distinct colours and burst them up. He can use the devices as many times as he wish.

Soon, Chef started to find the game boring. So, he wants to finish the game as soon as possible. Find out how many minimum number of times Chef has to use the devices?

Input

The first line of the input contains an integer T denoting the number of test cases. The description of T test cases follows.

For each test case, first line consists the string s denoting the configuration of the colors of balloons Chef's sister bought..

Second line of each test case, contains an integer K, as denoted above.

Output

For each test case, output a single line containing an integer corresponding to the answer of the problem.

Constraints

  • 1 ≤ T ≤ 10
  • 1 ≤ n = |s| ≤ 50
  • Each character of s is a lower case English alphabet, i.e. 'a' ≤ si ≤ 'z'.
