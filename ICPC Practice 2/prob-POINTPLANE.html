You are given n distinct points on a 2-D plane. You want to find whether there exists a line which pass through exactly two points out of these n points. Output "yes" or "no" accordingly.

Input

First line of the input contains an integer T denoting the number of test cases. T test cases follow.

First line of each test case contains an integer n.

Each of the next n lines contains two xi, yi denoting coordinates of i-th point.

Output

For each test case, output a single line containing "yes" or "no" (without quotes).

Constraints

  • 1 ≤ T ≤ 10
  • n ≤ T ≤ 105
  • -109 ≤ xi, yi ≤ 109
  • All the points have distinct coordinates

Example

Input
2
2
0 0
1 1
3
0 0
1 1
2 2

Output:
yes
no
